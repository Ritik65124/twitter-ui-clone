# Twitter UI Clone (Tailwind CSS)

This project is a **UI-only clone of Twitter** built using **Tailwind CSS**.
It was created purely for **learning and practice purposes**, focusing on
layout, responsiveness, and utility-first CSS.

## Tech Stack
- HTML
- Tailwind CSS

## Project Scope
- UI only (no JavaScript, no backend)
- Static layout
- Responsive design
